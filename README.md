# Wobb Influencer Discovery

A modern influencer discovery application built with React, TypeScript, Vite, Tailwind CSS, and Zustand.

## Features

- Search influencers
- Filter by platform
- Profile details page
- Add profiles to a persistent list
- Remove profiles
- Clear all selected profiles
- Duplicate prevention
- Persistent state using Zustand
- Responsive UI

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Zustand
- React Router

## Getting Started

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```
